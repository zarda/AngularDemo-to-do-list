# AngularDemo-to-do-list
As a demonstration to build frontend from scratch

# How to start

`> cd client`

`> ng serve`

# How to browse this
http://localhost:4200/

# How to build it
`> cd client`

`> ng build`

# How to test it
`> cd client`

`> ng test --code-coverage`

# Trouble shooting
## Before build you may need to install libs
`> yarn install`
